# timenow
Show current time whereever

## Installation
```bash
npm i timenow -g
```

## Usage
```bash
$ timenow
8:54:46 am, November 2nd 2016

$ timenow LAX
Los Angeles International Airport, 1 World Way, Los Angeles, CA 90045, USA:
12:54:49 am, November 2nd 2016 — (UTC-08:00) Pacific Time (US & Canada)

$ timenow la
12:54:54 am, November 2nd 2016 — (UTC-08:00) Pacific Time (US & Canada)

$ timenow pst
12:55:07 am, November 2nd 2016 — (UTC-08:00) Pacific Time (US & Canada)
Also found:
4:55:07 am, November 2nd 2016 — (UTC-04:00) Asuncion
12:55:07 pm, November 2nd 2016 — (UTC+05:00) Islamabad, Karachi
12:55:07 am, November 2nd 2016 — (UTC-08:00) Baja California
4:55:07 am, November 2nd 2016 — (UTC-04:00) Asuncion
12:55:07 pm, November 2nd 2016 — (UTC+05:00) Islamabad, Karachi
```
